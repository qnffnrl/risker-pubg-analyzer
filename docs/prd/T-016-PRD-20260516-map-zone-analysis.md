# T-016 PRD — 맵 & 존 분석 (신규 기능)

> 상태: 📋 대기 | 생성일: 2026-05-16 | 유형: feat

## 목표

플레이어가 어느 맵에서 얼마나 잘 플레이하는지, 어느 지역에 랜딩하는지 분석한다.
맵별 성적 차이를 보여줘 "이 맵에서 강한 플레이어"임을 어필할 수 있게 한다.

## 분석 항목

### 맵별 성적
각 맵별로:
- 플레이 횟수
- 평균 킬 / 데미지 / 순위
- 승률
- 강점 맵 자동 감지 ("에란겔 특화")

### 선호 랜딩존 (텔레메트리 기반, 가능 시)
- 텔레메트리 API로 초기 낙하 위치 파악
- 지역별 빈도: 학교 / 포체니키 / 밀리터리 베이스 등
- 핫 존 선호도 vs 안전 존 선호도

### 존 이동 패턴
- 평균 이동 시 원 단계 (서클 대비 위치)
- 엔딩 존 진입율 (상위 10명 남았을 때 생존율)
- 차량 이동 vs 도보 이동 비율 (맵별)

## UI 구성

```

승률: 11%        ★ 가장 강한 맵
```


## 데이터 소스

- 기본 매치 스탯: `map_name` 필드로 맵별 분리
- 랜딩존: 텔레메트리 API (호출 비용 있음, 캐싱 필수)
- 텔레메트리 없을 경우: 맵별 집계만 제공
