# T-015 PRD — 무기 & 교전 분석 (신규 기능)

> 상태: 📋 대기 | 생성일: 2026-05-16 | 유형: feat

## 목표

플레이어가 어떤 무기를 선호하고 어떻게 교전하는지 상세하게 분석·시각화한다.
"이 플레이어는 어떤 총을 주로 쓰는가?"는 PUBG 커뮤니티에서 가장 궁금해하는 지표 중 하나다.

## 분석 항목

### 무기 선호도
- 사용 무기 Top 10 (킬 수 기준)
- 무기별: 킬, 데미지, 헤드샷율, 평균 교전 거리 (estimated)
- 무기 카테고리 분포: AR / SR / SMG / DMR / Shotgun / Pistol
- "주력 무기 조합" 자동 감지 (예: M416 + AWM = 스나이퍼 콤보)

### 교전 패턴
- 킬 거리 분포 (근거리/중거리/장거리 비율)
- 헤드샷 비율 추이
- 선제 교전율 vs 반응 교전율 (추정, 제한적)

## 데이터 소스

PUBG API `player_match_stats.raw_stats`에서 무기 데이터 파싱:
- `WinPlace`, `KillStreaks`, `DBNOs` 등 활용
- 무기별 킬 데이터: 매치 텔레메트리(Telemetry) API 활용 (선택적, Rate Limit 주의)
- 텔레메트리 없을 경우: 매치 기본 스탯 + 무기 마스터리 API 병합
