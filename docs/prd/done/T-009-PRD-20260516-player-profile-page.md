# T-009 PRD — 플레이어 프로필 + 매치 목록 페이지

> 상태: 📋 대기 | 생성일: 2026-05-16 | 유형: feat

## 목표

특정 플레이어의 기본 정보와 최근 매치 목록을 보여주는 프로필 페이지를 구현한다.

## 페이지 구성

### URL
`https://pubg.risker.co.kr/players/{pubgId}`

### 상단: 플레이어 헤더
```
[대형 Avatar] [닉네임] [플랫폼 배지] [즐겨찾기 별 버튼]
             [마지막 분석 시각]
             [새로고침 버튼] → forceRefresh API 호출
```

### 요약 스탯 카드 (4개)
```
[킬/매치] [데미지/매치] [평균 순위] [게임 수]
```

### 플레이 스타일 미리보기 (→ T-010 대시보드로 이동 링크)
```
[미니 레이더 차트] [4대 성향 점수 바]
[상세 분석 보기 →]
```

### 매치 목록 (최근 20개)

각 매치 카드:
```
[맵명 + 모드] [플레이 일시]
[순위] [킬] [어시스트] [데미지] [생존시간]
[헤드샷] [이동거리] [부스터]
```

- 매치 모드별 컬러 배지: squad / duo / solo / FPP
- 순위 표시: 1위 = 골드 크라운 아이콘, 상위 10% = 민트, 나머지 회색
- 무한 스크롤 or 더보기 버튼 (추가 매치 로드)
- 매치 카드 클릭 → 해당 매치 상세 모달 (or 새 페이지 TBD)

### 시즌/모드 필터 (T-020과 연동)
```
[시즌 선택] [모드: 전체/스쿼드/듀오/솔로] [맵 선택]
```

// Next.js App Router
