# T-017 PRD — 성과 추이 차트 (신규 기능)

> 상태: 📋 대기 | 생성일: 2026-05-16 | 유형: feat

## 목표

최근 매치들의 시계열 데이터로 플레이어 성과 변화 추이를 시각화한다.
"최근 실력이 늘고 있는지 줄고 있는지"를 한눈에 파악할 수 있게 한다.

## 분석 항목

### 이동 평균 지표 (최근 5/10/20게임)
- 킬 추이 라인 차트
- 데미지 추이 라인 차트
- 순위 추이 라인 차트

### 폼 분석
- 최근 5게임 vs 이전 15게임 비교
- 상승/하강/유지 트렌드 감지
- 연승/연패 스트릭 감지

### 성향 점수 변화
- 분석 시점별 4대 성향 점수 추이 (여러 번 분석된 경우)

## UI 구성

### 성과 추이 탭 (플레이어 프로필 내)

**메인 라인 차트**
```
킬수  5 |        /\
      4 |   /\  /  \
      3 |  /  \/    \___
      2 | /
      1 |/
        |-----------→ 매치 (최근 20게임)
        
토글: [킬] [데미지] [순위] [생존시간]
기간: [최근 10게임] [최근 20게임] [전체]
```

**폼 카드**
```
[최근 폼] 📈 상승세
최근 5게임 평균 킬: 4.2  (이전 대비 +1.1)
최근 5게임 평균 데미지: 320  (이전 대비 +45)
```

**연속 기록 배지**
```
🔥 현재 3게임 연속 킬 5개 이상
🏆 최근 10게임 TOP 10 진입: 7회
```

**핫 스트릭 / 슬럼프 감지**
```typescript
// 최근 3게임 킬 평균이 전체 평균의 150% 이상 → "핫 스트릭"
// 최근 3게임 킬 평균이 전체 평균의 50% 이하 → "슬럼프"
```
