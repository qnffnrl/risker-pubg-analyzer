# T-010 PRD — 플레이 스타일 분석 대시보드

> 상태: 📋 대기 | 생성일: 2026-05-16 | 유형: feat

## 목표

플레이어의 4대 성향 분석 결과를 시각화하는 핵심 대시보드를 구현한다.
이 페이지가 앱의 핵심 가치를 전달하는 메인 화면이다.

## 페이지 구성

### URL
`http://risker.co.kr/risker-pubg-analyzer/players/{pubgId}/analysis`
(또는 `/players/{pubgId}` 탭으로 통합 가능)

### 레이아웃 (데스크톱 — 2컬럼)

**좌측 (40%)**
- 대형 레이더 차트 (4축: 공격성/생존/포지셔닝/팀플)
- 각 축 점수 숫자 표시 (0~100)
- 차트 아래: 성향 요약 레이블 (예: "공격적 생존형")
- LLM 자연어 요약 텍스트 박스 (T-011)

**우측 (60%)**
- 4대 성향별 상세 지표 섹션 (아코디언 or 탭)
- 각 섹션: 해당 지표 카드 그리드

### 레이더 차트 (핵심 컴포넌트)

```
       공격성(Aggression)
          /\
         /  \
포지셔닝 ---- 팀플레이
         \  /
          \/
        생존형(Survival)
```

- recharts `RadarChart` 사용
- 배경: 다크 + 포인트 컬러 그라데이션 fill
- 애니메이션: 진입 시 그려지는 효과
- 호버: 각 꼭짓점에 툴팁 (점수 + 상위 N%)

### 4대 성향 상세 카드

**공격성 섹션**
```
[킬/매치: 3.2] [데미지/매치: 285] [헤드샷율: 31%] [교전참여율: 42%]
```

**생존형 섹션**
```
[평균 생존시간: 20:40] [평균 순위: 8.3위] [탑10율: 45%] [승률: 8%]
[평균 부스터: 3.1] [평균 힐: 4.2]
```

**포지셔닝 섹션**
```
[도보 이동: 2,840m] [차량 이동: 1,200m] [차량 사용율: 65%]
[평균 무기수: 5.1] [선호 맵: 에란겔]
```

**팀플레이 섹션**
```
[평균 부활: 0.8회] [평균 어시스트: 1.8회] [팀킬: 0.02회]
```

### 성향 레이블 시스템

4대 점수 조합으로 자동 레이블 생성:
```typescript
// 예시 규칙
if (aggression > 70 && survival < 40) → "러셔 (Rusher)"
if (survival > 70 && positioning > 70) → "존버형 (Zone Hugger)"
if (aggression > 70 && teamplay > 70) → "팀 에이스 (Team Ace)"
if (positioning > 70 && survival > 70 && aggression < 40) → "서클 마스터"
// ... 16개 조합 레이블
```

레이블 UI:
```
[아이콘] "러셔 (Rusher)"
"교전을 즐기며 공격적으로 킬을 추구합니다"
```
